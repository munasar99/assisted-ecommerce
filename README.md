# Ubaxsana — Assisted E-commerce (Full Stack)

```
assisted-ecommerce/
├── frontend/          React + Tailwind + API integration
└── backend/
    └── AssistedEcommerce.Api/   ASP.NET Core + MongoDB (ubaxsana)
```

## Quick start

1. **MongoDB** running (local or Atlas)
2. **Backend:** `cd backend/AssistedEcommerce.Api` → `dotnet run`
3. **Frontend:** `cd frontend` → `npm install` → `npm run dev`

| App | URL |
|-----|-----|
| React | http://localhost:5173 |
| API Swagger | http://localhost:5298/swagger |
| MongoDB test | http://localhost:5298/api/health/mongodb |

## Admin
