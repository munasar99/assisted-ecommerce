Base URL: http://localhost:5298  |  Swagger  |  Health

Qaladka sawirkaaga (Invalid URL):
Dropdown-ka wuxuu ahaa GET laakiin URL-ka waxaad ku qortay PUT http://localhost:5298/...

✓ Sax: Dropdown-ka dooro PUT — URL-ka geli kaliya: http://localhost:5298/api/orders/ORD-001-03 (erayga PUT ha ku jirin URL-ka dhexdiisa!)
1. API kici: dotnet run gudaha AssistedEcommerce.Api
2. Boggan fur: http://localhost:5298/api-kbaro
3. Fur API-KBARO.http — riix "Send Request" (JSON diyaar)

Thunder Client — 4 goobood

GoobMaxaa gelisaa
1. Method (dropdown)GET, POST, PUT, ama DELETE — hal dooro
2. URL (bar)Kaliya http://localhost:5298/api/... — MA ku dar GET/PUT halkan
3. Body tabPOST iyo PUT: JSON ON. GET iyo DELETE: Body OFF / madhan
4. Auth tabAdmin: Login → Bearer token (waa lagama maarmaan)

POST Order geli (data cusub)

URL: http://localhost:5298/api/orders

Body: JSON ON — koobi-geli:

{
  "fullName": "jaamac ali",
  "phone": "+25267889855",
  "productUrl": "https://www.alibaba.com/example",
  "productName": "miis",
  "productUnitPriceUsd": 20,
  "quantity": 4,
  "deliveryType": "HomeDelivery",
  "districtId": "ZONE-HODAN",
  "addressDetail": "abdicasiis, Muqdisho"
}

Pickup: "deliveryType": "Pickup" — ha ku darin districtId

Jawaabta ka qor orderId (tusaale ORD-001-03) — talaabada xigta!

GET Liiska orders (akhri)

URL: http://localhost:5298/api/orders?page=1

Body: madhan / OFF

GET Hal order

URL: http://localhost:5298/api/orders/ORD-001-03

Qalad: .../orders/{orderId}  |  Sax: ID dhab ah

Body: madhan

PUT Beddel order

  1. Dropdown: dooro PUT (ma aha GET)
  2. URL: http://localhost:5298/api/orders/ORD-001-03 — PUT ha ku qorin!
  3. Body JSON (waxaad beddeshay oo keliya):
{
  "productName": "miis cusub",
  "quantity": 4,
  "notes": "updated"
}

Qalad: safka 1aad POST http://localhost:5298/api/orders — JSON ma noqon karo!

Pickup (Tabo): PUT — { "deliveryType": "Pickup" } — districtId ma loo baahna

Home delivery: PUT — { "deliveryType": "HomeDelivery", "districtId": "ZONE-HODAN" }

DELETE Tirtir order

URL: http://localhost:5298/api/orders/ORD-001-03

Body: madhan
